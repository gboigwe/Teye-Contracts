## Quick Start Checklist

1. Install Rust 1.78.0 and the required toolchain components.
2. Install Soroban CLI v23.1.4 and verify the version.
3. Run `./setup.sh` or `make setup`.
4. Run `make test` to verify your environment.
5. Read the codebase tour and pick a good first issue.
