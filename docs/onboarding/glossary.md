# Glossary

This glossary covers project, domain, and blockchain terminology.

## Soroban and Rust

- Contract - A deployed WASM program running on Soroban.
- ContractError - A typed error returned by contract methods.
- Env - Soroban execution environment handle.
- Symbol - Compact symbol type used for keys and identifiers.
- Bytes / BytesN - Binary data containers used in Soroban.
- Storage - Persistent, instance, or temporary data storage.
- TTL - Time-to-live for storage entries.
- Ledger - The current chain state snapshot.
- RPC - Remote procedure call endpoint for Soroban.
- XDR - External Data Representation used by Stellar.
- WASM - WebAssembly contract binary format.
- wasm32-unknown-unknown - Rust target for Soroban contracts.
- rust-analyzer - Rust language server used by IDEs.
- Clippy - Rust linter.
- rustfmt - Rust formatter.

## Healthcare and Vision Care

- PHI - Protected Health Information.
- HIPAA - US health privacy regulation.
- FHIR - Fast Healthcare Interoperability Resources standard.
- EMR - Electronic Medical Records.
- EHR - Electronic Health Records.
- Optometry - Primary vision care and eye exams.
- Ophthalmology - Medical and surgical eye care.
- Prescription - Lens or treatment specification for patients.
- Visual acuity - Clarity or sharpness of vision.
- Intraocular pressure - Pressure inside the eye.
- Slit lamp exam - Microscopic eye examination method.
- Fundus photography - Imaging of the interior eye surface.
- Retinal imaging - Imaging of the retina and optic nerve.
- Contact lens fitting - Determining appropriate lens parameters.

## Cryptography and Privacy

- ZK proof - Zero-knowledge proof for privacy-preserving verification.
- Verifier - Contract or system that checks a proof.
- Prover - System that generates a proof.
- Homomorphic encryption - Computation on encrypted data.
- Differential privacy - Noise-based privacy preservation.
- Key rotation - Periodic replacement of cryptographic keys.
- Multisig - Multiple signature authorization policy.
- Access policy - Rules that determine who can access data.
- RBAC - Role-based access control.
- Consent grant - Authorization from a patient for access.
- Audit trail - Immutable log of access events.

## Blockchain and Governance

- Contract upgrade - Deploying a new contract version.
- Migration - Transforming stored data to a new format.
- Governance - On-chain decision-making processes.
- Timelock - Delayed execution of governance actions.
- Staking - Locking assets to secure or govern the protocol.
- Treasury - Funds reserved for protocol operations.
- Ledger entry - A stored value on-chain.
- Network passphrase - Network identifier for Stellar.
- Testnet - Public testing network.
- Mainnet - Production Stellar network.
- Futurenet - Staging network for upcoming features.
